# Ainulindale API

Public application source repository for the Ainulindale API project.

This repository is intended to demonstrate a small, presentable, JSON-only API
service that can later be built, tested, packaged, and deployed through a
GitHub-based CI/CD and GitOps flow.

## Current status

Chunk 1 establishes the repository baseline only:

- public repository visibility
- minimal documentation
- contribution notes
- license
- minimal ignore rules
- dependency update configuration
- GitHub-side ruleset documentation
- protected `main` workflow

No production API implementation is expected in this chunk.

## Intended future direction

The future service is expected to use FastAPI and expose public application
endpoints that receive JSON and return JSON.
